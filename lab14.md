```
int a[9]={60,65,75,80,90,95};
int x[]={50,67,99};
int k=0;
int i,j;
for(i=0;i<5;i++){
    if(x[k]>a[i]&&x[k]<a[i+1]){
        for(j=6+k;j>i+1;j--){
            a[j]=a[j-1];
        }
        a[i+1]=x[k];
        k++;
    }
    else if(x[k]<a[0]){
        for(j=6+k;j>0;j--){
            a[j]=a[j-1];
        }
        a[0]=x[k];
        k++;
    }
    else if(x[k]>a[5+k]){
        a[6+k]=x[k]
        k++;
    }
}
```
## 思考：选择插入点在list头上、中间、尾巴上的三个数作为算法测试的数据，是从极端的方式来测试程序的可行性，能更好的检测程序是否完美。
